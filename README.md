# ml-neural-networks

Neural network implementations for character-level name generation, following Andrej Karpathy's "makemore" series.

## Notebooks

- **Bigram MakeMore.ipynb** - Simple bigram language model using character pair statistics
- **MLP MakeMore.ipynb** - Multi-layer perceptron model with embeddings for name generation

## Data

- `names.txt` - Training dataset of names
